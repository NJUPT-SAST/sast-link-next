SAST Link 是南京邮电大学学生科协（SAST）的统一身份与账号管理平台。本站记录了 `sast-link-next` 的完整实现细节：基于 Next.js 16 构建的 Web 应用，同时支持 Tauri 2 桌面端打包。

## 技术栈

| 层级 | 技术 |
|---|---|
| 框架 | Next.js 16，App Router |
| 语言 | TypeScript 5 |
| 样式 | Tailwind CSS v4，shadcn/ui 风格组件 |
| 状态 | Zustand 5，SWR 2 |
| HTTP | Axios 1 |
| 桌面端 | Tauri 2 |
| 测试 | Jest 30，Testing Library，MSW 2 |
| 校验 | Zod 4，React Hook Form 7 |
