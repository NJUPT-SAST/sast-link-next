## 路径别名

TypeScript 路径别名 `@/*` 映射至仓库根目录。所有内部导入均应使用此别名：

```ts
import { cn } from "@/lib/utils";
import { useAuthStore } from "@/store/use-auth-store";
import { Button } from "@/components/ui/button";
```
