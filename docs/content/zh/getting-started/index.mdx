## 环境要求

| 依赖 | 版本 | 用途 |
|---|---|---|
| Node.js | 20 及以上 | JavaScript 运行时 |
| pnpm | 9 及以上 | 包管理器 |
| Rust 工具链 | stable | 仅 Tauri 桌面构建需要 |
| Tauri CLI 系统依赖 | 因操作系统而异 | 仅 Tauri 桌面构建需要 |

Tauri 所需的系统依赖（Windows 上的 WebView2、Linux 上的 webkit2gtk、macOS 上的 Xcode），请参阅 [Tauri 前置依赖指南](https://tauri.app/start/prerequisites/)。

## 克隆与安装

```bash
git clone https://github.com/SAST-Link/sast-link-next.git
cd sast-link-next
pnpm install
```

## 环境变量

在仓库根目录创建 `.env.local` 文件：

```bash
# 必填：后端 API 服务器的基础 URL
NEXT_PUBLIC_API_BASE_URL=https://api.example.com

# 可选：启用 MSW 浏览器模拟（无真实后端时设为 "true"）
NEXT_PUBLIC_API_MOCKING=true
```

### 变量说明

| 变量 | 是否必填 | 说明 |
|---|---|---|
| `NEXT_PUBLIC_API_BASE_URL` | 是 | 所有 Axios 请求的基础 URL，末尾不加斜杠。 |
| `NEXT_PUBLIC_API_MOCKING` | 否 | 设为 `"true"` 时，`app/providers.tsx` 会在浏览器中启动 MSW Service Worker，拦截所有 API 调用并转发至 `mocks/handlers/` 中的模拟处理器。适用于本地开发及 Jest 测试环境。 |

## 启动开发服务器

```bash
pnpm dev
```

应用将在 `http://localhost:3000` 启动。任何对 `app/`、`components/`、`lib/`、`store/` 或 `hooks/` 下文件的修改都会触发 Fast Refresh 热更新。
