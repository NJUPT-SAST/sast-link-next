所有函数均使用 `lib/api/client.ts` 中的共享 `apiClient`。

## verifyRegistAccount

```ts
function verifyRegistAccount(username: string): Promise<AxiosResponse<ResType<{ registerTicket: string }>>>
```

检查指定用户名是否可用于注册。

- **方法**：`GET /verify/account`
- **查询参数**：`username`、`flag=0`
- **返回**：`registerTicket`，后续所有注册请求均需携带。

## verifyLoginAccount

```ts
function verifyLoginAccount(username: string): Promise<AxiosResponse<ResType<{ loginTicket: string }>>>
```

检查指定用户名是否存在以供登录。

- **方法**：`GET /verify/account`
- **查询参数**：`username`、`flag=1`
- **返回**：`loginTicket`，下一步传入 `userLogin`。

## verifyResetAccount

```ts
function verifyResetAccount(username: string): Promise<AxiosResponse<ResType<{ resetPwdTicket: string }>>>
```

检查指定用户名是否存在以供密码重置。

- **方法**：`GET /verify/account`
- **查询参数**：`username`、`flag=2`
- **返回**：`resetPwdTicket`，后续所有重置请求均需携带。

## sendMail

```ts
function sendMail(ticket: string, type?: "reset"): Promise<AxiosResponse<ResType<null>>>
```

向与票据关联的邮箱发送验证邮件。

- **方法**：`GET /sendEmail`
- **请求头（注册）**：`REGISTER-TICKET: <ticket>`
- **请求头（重置）**：`RESETPWD-TICKET: <ticket>`（当 `type === "reset"` 时）
- **返回**：成功时返回 `null`。

在 `verifyRegistAccount` 或 `verifyResetAccount` 之后调用，用户点击"重新发送"时也会再次调用。

## verifyCaptcha

```ts
function verifyCaptcha(
  ticket: string,
  captcha: string,
  type?: "reset",
): Promise<AxiosResponse<ResType<null>>>
```

验证用户输入的 5 位验证码，内部发送前自动加上 `"S-"` 前缀。

- **方法**：`POST /verify/captcha`
- **Content-Type**：`application/x-www-form-urlencoded;charset=UTF-8`
- **请求体**：`captcha=S-<code>`
- **请求头（注册）**：`REGISTER-TICKET: <ticket>`
- **请求头（重置）**：`RESETPWD-TICKET: <ticket>`
- **返回**：成功时返回 `null`。

## userRegister

```ts
function userRegister(password: string, registerTicket: string): Promise<AxiosResponse<ResType<null>>>
```

创建新用户账号。

- **方法**：`POST /user/register`
- **Content-Type**：`application/x-www-form-urlencoded`
- **请求体**：`password=<password>`
- **请求头**：`REGISTER-TICKET: <registerTicket>`
- **返回**：成功时返回 `null`。

在注册流程 `verifyCaptcha` 成功后调用。

## resetPassword

```ts
function resetPassword(password: string, resetTicket: string): Promise<AxiosResponse<ResType<null>>>
```

为已有账号设置新密码。

- **方法**：`POST /user/resetPassword`
- **Content-Type**：`application/x-www-form-urlencoded`
- **请求体**：`newPassword=<password>`（注意：字段名与 `userRegister` 不同）
- **请求头**：`RESETPWD-TICKET: <resetTicket>`
- **返回**：成功时返回 `null`。

## getFeishuLoginStatus

```ts
function getFeishuLoginStatus(code: string, state: string): Promise<AxiosResponse<any>>
```

将飞书 OAuth 授权码换取会话令牌或 OAuth 票据。

- **方法**：`GET /login/lark/callback`
- **查询参数**：`code`、`state`
- **返回**：`{ loginToken }`（直接登录）或 `{ oauthTicket }`（需账号绑定）。

由 `app/(tourist)/callback/feishu/page.tsx` 调用。

function getGithubLoginStatus(code: string, state: string): Promise<AxiosResponse<any>>
