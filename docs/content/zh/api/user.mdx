所有函数均使用 `lib/api/client.ts` 中的共享 `apiClient`。已认证端点依赖 Axios 请求拦截器注入的 `Token` 请求头。

## userLogin

```ts
function userLogin(
  password: string,
  loginTicket: string,
  oauthTicket?: string | null,
): Promise<AxiosResponse<ResType<{ loginToken: string }>>>
```

验证用户身份并返回会话令牌。

- **方法**：`POST /user/login`
- **请求体**：`multipart/form-data`，字段名 `password`
- **请求头**：`LOGIN-TICKET: <loginTicket>`
- **请求头（可选）**：`OAUTH-TICKET: <oauthTicket>`，仅在 OAuth 账号绑定流程时存在
- **返回**：`loginToken`，通过 `setToken(token)` 持久化至 localStorage。

## getUserInfo

```ts
function getUserInfo(): Promise<AxiosResponse<ResType<{ email: string; userId: string }>>>
```

返回已认证用户的基础身份信息。

- **方法**：`GET /user/info`
- **认证**：需要 `Token` 请求头
- **返回**：`email` 和 `userId`。

## userLogout

```ts
function userLogout(): Promise<AxiosResponse<ResType<null>>>
```

在后端使当前会话失效。

- **方法**：`POST /user/logout`
- **认证**：需要 `Token` 请求头
- **返回**：成功时返回 `null`。

调用后，通过 `clearToken()` 从 localStorage 移除令牌，并通过 `logout()` 重置 `useAuthStore`。

## getUserProfile

```ts
function getUserProfile(): Promise<AxiosResponse<ResType<UserProfileType>>>
```

返回已认证用户的完整资料。

- **方法**：`GET /profile/getProfile`
- **认证**：需要 `Token` 请求头
- **返回**：`UserProfileType`，类型定义见 [API 客户端](/zh/docs/api)。

这是应用中唯一的资料拉取操作，由 `hooks/use-fetch-profile.ts` 中的 `useFetchProfile()` 调用，并将结果写入所有相关 Zustand Store。

## editProfile

```ts
function editProfile(data: EditableProfileType): Promise<AxiosResponse<ResType<null>>>
```

更新已认证用户的资料字段。

- **方法**：`POST /profile/changeProfile`
- **认证**：需要 `Token` 请求头
- **请求体**：JSON，包含 `EditableProfileType` 的任意子集字段
- **返回**：成功时返回 `null`。

调用成功后，使用 `useUserProfileStore.updateProfile(fields)` 同步本地 Store，保持 UI 数据一致。

## uploadAvatar

```ts
function uploadAvatar(file: Blob): Promise<AxiosResponse<ResType<{ filePath: string }>>>
```

上传新的头像图片。

- **方法**：`POST /profile/uploadAvatar`
- **认证**：需要 `Token` 请求头
- **请求体**：`multipart/form-data`，字段名 `avatarFile`
- **返回**：`filePath`，即存储后的头像图片 URL。

`Blob` 通常由 `app/(user)/home/edit/page.tsx` 中的 `react-avatar-editor` 画布导出生成。

## getUserBindStatus

```ts
function getUserBindStatus(): Promise<AxiosResponse<ResType<string[]>>>
```

返回已关联至当前用户账号的 OAuth 服务商标识列表。

- **方法**：`GET /profile/bindStatus`
- **认证**：需要 `Token` 请求头
- **返回**：服务商名称字符串数组（如 `["github", "feishu"]`）。

由 `BindAppItem` 组件在资料 UI 中展示已绑定账号。
