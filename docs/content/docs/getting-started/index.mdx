## Prerequisites

| Requirement | Version | Purpose |
|---|---|---|
| Node.js | 20 or later | JavaScript runtime |
| pnpm | 9 or later | Package manager |
| Rust toolchain | stable | Required only for Tauri desktop builds |
| Tauri CLI prerequisites | varies by OS | Required only for Tauri desktop builds |

For Tauri-specific system dependencies (WebView2 on Windows, webkit2gtk on Linux, Xcode on macOS), refer to the [Tauri prerequisites guide](https://tauri.app/start/prerequisites/).

## Clone and install

```bash
git clone https://github.com/SAST-Link/sast-link-next.git
cd sast-link-next
pnpm install
```

## Environment variables

Create a `.env.local` file in the repository root:

```bash
# Required: base URL of the backend API server
NEXT_PUBLIC_API_BASE_URL=https://api.example.com

# Optional: enable MSW browser mocking (set to "true" for local development without a real backend)
NEXT_PUBLIC_API_MOCKING=true
```

### Variable reference

| Variable | Required | Description |
|---|---|---|
| `NEXT_PUBLIC_API_BASE_URL` | Yes | Base URL prepended to every Axios request. Must not have a trailing slash. |
| `NEXT_PUBLIC_API_MOCKING` | No | When set to `"true"`, `app/providers.tsx` bootstraps the MSW service worker in the browser, intercepting all API calls with the mock handlers in `mocks/handlers/`. Useful for local development and in the Jest test environment. |

## Start the development server

```bash
pnpm dev
```

The application starts at `http://localhost:3000`. Hot module replacement is active; any file change in `app/`, `components/`, `lib/`, `store/`, or `hooks/` triggers a Fast Refresh update.
