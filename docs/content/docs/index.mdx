SAST Link is the identity and account management platform for SAST (Software Association of Science and Technology). This site documents the `sast-link-next` implementation: a Next.js 16 web application with Tauri 2 desktop packaging.

## Technology overview

| Layer | Technology |
|---|---|
| Framework | Next.js 16, App Router |
| Language | TypeScript 5 |
| Styling | Tailwind CSS v4, shadcn/ui-style components |
| State | Zustand 5, SWR 2 |
| HTTP | Axios 1 |
| Desktop | Tauri 2 |
| Testing | Jest 30, Testing Library, MSW 2 |
| Validation | Zod 4, React Hook Form 7 |
